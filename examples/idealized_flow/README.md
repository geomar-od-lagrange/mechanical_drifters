# Drogued Drifter in Idealized Flow

Demonstrates the core drogued drifter physics in synthetic velocity fields.

| Notebook | Description |
|----------|-------------|
| `01_synthetic_flow_profiles.ipynb` | Drogued drifter in a sheared profile; intermediate speed between surface and drogue depth |
| `02_sheared_jet_parcels.ipynb` | Same physics coupled to Parcels v4 for Lagrangian particle tracking |
| `03_drogued_drifter_in_wave_orbitals.ipynb` | Stokes drift reference levels: Lagrangian mechanics derivation comparing surface tracer, deep tracer, extended drogued drifter in wave orbitals, and production `DroguedDrifter` in a Stokes profile |

**Requirements**: Parcels v4, numpy, scipy, sympy, matplotlib, xarray
**Runtime**: ~2–5 minutes (01, 03), ~5–10 minutes (02)
